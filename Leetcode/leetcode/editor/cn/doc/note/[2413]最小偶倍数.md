1. 奇偶性判断
n & 1 使用与操作来判断正整数 n 的奇偶性.
主要是看 n 的最后一位是否为1：
    若最后一位为1，则 & 操作，结果为 1（奇数）, 否则值为 0（偶数）。
2. 移位操作
左移操作（`<<`）：乘法
操作： A << B
数学等价： 将一个数 A 左移 B 位。等同于计算 A * 2^B。

右移操作：（`>>`）：除法
等同于计算 A / 2^B。
通常用于执行**整数除法**，并且处理**负数**时有两种不同的类型：
- 逻辑右移（`>>>`）：无论整数是正数还是负数，都将其右移。
  行为：左边总是补0。
  对于正数，等同于 A / 2^B，结果向下取整（截断小数部分）。
- 算术右移（`>>`）：仅当整数为正数时才将其右移。
  行为：左边补的是`符号位`，(（即如果原数是正数就补`0`，如果原数是负数就补`1`)。 
  对于正数，与逻辑右移一样，等同于 `A / 2^B`，结果向下取整。
  对于负数，算术右移更接近于向负无穷大取整的除法。
  例如，在 Go 中 `-10 >> 1 = -5`，这符合数学除法 `[-10/2] = -5`。
  但在某些语言（如 C）中，负数右移的行为依赖于实现，但通常也是为了保持数值的符号。
